# Tener - Web3 Growth Portfolio

A modern, professional portfolio website showcasing Web3 growth expertise and case studies.

## Features

- **Modern Design**: Built with Next.js 14 and TailwindCSS
- **Responsive**: Optimized for all devices
- **Animations**: Smooth transitions with Framer Motion
- **Web3 Focused**: Designed for crypto founders and VCs
- **Case Study Driven**: Highlights real traction and metrics

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: TailwindCSS with custom dark theme
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Typography**: Inter & JetBrains Mono

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Customization

- Update profile information in `app/page.tsx`
- Modify colors in `tailwind.config.js`
- Add your own profile image in the hero section
- Update contact links and social profiles

## Deployment

This portfolio is ready to deploy on:
- Vercel (recommended)
- Netlify
- Any static hosting service

## Portfolio Sections

1. **Hero Section**: Name, title, and positioning
2. **Growth Philosophy**: Approach and methodology
3. **Case Studies**: TON House of Stars & Nest Wallet with metrics
4. **Skill Stack**: Core growth competencies
5. **Contact**: Call-to-action and links
